This project uses requests and BeautifulSoup to crawl stock price. In this project i have taken example of one stock, where multiple stock cab be scraped.
Steps to perform - 
1. Install all the packages mentioned in requirements.txt file.
2. Create account in twilio and get the Account SID and Auth Token and add that in stock.py file in line no 15 and 16.
3. in twilio site, navigate to programable SMS and than Whatsapp Message and verify your phone number.
4. apscheduler package is used to schedule this job everyday from 9:30am to 4:30pm IST every hours.
