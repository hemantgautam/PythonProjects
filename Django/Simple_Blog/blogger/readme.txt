This is a simple blog project with basic responsive HTML. In this project you will learn about below points - 
 - Url Mapping
 - Django Model Creation(ORM)
 - How to Register Model
 - Django generic ListView, DetailView
 - Pagination on Blog List view
 - Dummy responsive Html is added inside tempaltes folder. Html can be changed according to the project

Note: 
 - App name is "blog"
 - Run pthon manage.py migrate before using the project
