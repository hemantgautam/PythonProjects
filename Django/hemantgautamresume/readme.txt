This projects is to create simple resume website by using Django to under how Django works. Reference http://hemantgautamresume.me/
Its using static html file to show the content. its not using Django ORM(object relational mapper) concept, but can be done by simply adding all
the fields in model. App name is "myresume". Install all the packages mentioned in requirements.txt file and after that it should work on local
environment.
